# Amsterdam Home Exchange Portfolio

A modern, multi-page website for presenting an Amsterdam apartment for home exchange opportunities.

## Features

✨ **Modern Design**
- Contemporary typography and color palette
- Responsive navigation with mobile menu
- Clean, professional layout

📸 **Dynamic Gallery**
- Automatic photo detection from `/photos` folder
- Lightbox viewer with keyboard navigation
- Works with any number of photos

📄 **Multi-Page Structure**
- Home page with highlights
- Gallery page with full photo showcase
- Neighborhood information
- Exchange preferences
- Contact page

🔒 **Privacy-Focused**
- No exact address displayed
- No external tracking
- Clean, static HTML

## Pages

- **index.html** - Home page with apartment highlights
- **gallery.html** - Full photo gallery with lightbox
- **neighborhood.html** - Neighborhood information
- **exchange.html** - Exchange preferences
- **contact.html** - Contact information

## Getting Started

1. Upload photos to the `/photos` folder
2. Edit text sections marked with `<!-- EDIT -->`
3. Update contact email
4. Deploy to GitHub Pages

## Photo Setup

Please add your photos to the `/photos` folder. The gallery will automatically detect and display all images (jpg, jpeg, png, gif, webp).

## Customization

Edit these files to customize:
- Text content: Edit HTML files directly
- Colors/styling: Edit `style.css`
- Functionality: Edit `script.js` and `gallery-loader.js`

## Navigation

- Mobile-responsive hamburger menu
- Active page highlighting
- Smooth scrolling

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
